# Аналіз відкритих даних: Реєстрація транспортних засобів в Україні

**Мета проєкту:** Розробити конвеєр обробки даних (від завантаження до візуалізації) для аналізу інформації про реєстрацію транспортних засобів (МВС) з метою виявлення ринкових трендів та перевірки якості державних даних.

**Джерело відкритих даних:** [Відомості про транспортні засоби та їх власників (data.gov.ua)](https://data.gov.ua/)

## 3 питання/гіпотези для аналізу:

1. **Гіпотеза 1 (Монополія брендів):** Топ-3 найпопулярніші марки авто (`BRAND`) займають понад 30% усього зареєстрованого автопарку в цьому датасеті.
2. **Гіпотеза 2 (Вік автопарку):** Більше половини зареєстрованих автомобілів у базі мають вік старше 10 років (розраховується на основі колонки `MAKE_YEAR`).
3. **Гіпотеза 3 (Якість даних та аномалії):** У державному реєстрі присутні помилкові або аномальні записи.
