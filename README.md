# Аналіз відкритих даних: Реєстрація нового бізнесу (ФОП)

**Мета проєкту:** Аналіз динаміки відкриття нових бізнесів (ФОП) за регіонами та сферами діяльності для розуміння економічних трендів.

**Джерело відкритих даних:** [Національний портал відкритих даних](https://data.gov.ua/)

## 3 питання/гіпотези для аналізу:
1. **Гіпотеза 1:** Сфера IT (комп'ютерне програмування) має найбільший відсоток реєстрацій серед молоді (вікова категорія 18-25 років).
2. **Гіпотеза 2:** У великих містах (Київ, Львів, Дніпро) кількість реєстрацій нового бізнесу значно зростає у весняні місяці.
3. **Гіпотеза 3:** Відсоток жінок, які відкривають власний бізнес у сфері роздрібної торгівлі та послуг краси, значно перевищує відсоток чоловіків у цих сферах.Рядок з гілки Б
